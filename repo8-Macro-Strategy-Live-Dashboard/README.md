# 智能投资分析专家仪表板

一个基于实时数据和AI分析的综合投资决策支持系统，整合了市场概览、投资组合管理、深度分析和AI智能洞察功能。

## 🌟 主要功能

### 📊 实时市场数据
- **主要指数**: 标普500、纳斯达克、道琼斯等
- **商品价格**: 黄金、原油等大宗商品
- **加密货币**: 比特币、以太坊等主流币种
- **经济指标**: 国债收益率、美元指数、VIX恐慌指数等

### 💼 投资组合管理
- **智能资产配置**: 基于风险偏好的个性化配置建议
- **风险评估**: 实时计算投资组合风险指标
- **具体建议**: 详细的ETF和个股推荐

### 🔍 深度市场分析
- **行业分析**: 各行业板块表现对比
- **技术指标**: RSI、MACD、移动平均线等
- **市场情绪**: 恐惧贪婪指数、资金流向分析

### 🤖 AI智能洞察
- **智能投资建议**: 基于Google AI的个性化投资建议
- **风险评估**: AI驱动的综合风险评估
- **异常检测**: 实时市场异常情况识别

## 🚀 快速开始

### 前置要求
- Node.js 16.0.0 或更高版本
- npm 8.0.0 或更高版本
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 安装步骤

1. **安装依赖**
   ```bash
   npm install
   ```

2. **启动服务**
   ```bash
   npm start
   ```

3. **访问界面**
   
   在浏览器中访问: `http://localhost:3000`

## 🎯 使用指南

### 基本使用
1. 启动系统后，打开浏览器访问 `http://localhost:3000`
2. 点击"刷新实时数据"获取最新市场数据
3. 浏览不同模块了解市场情况
4. 点击"生成AI投资建议"获取智能分析
5. 根据建议调整投资策略

### 功能模块
- **市场概览**: 查看实时市场数据和关键指标
- **投资组合**: 了解推荐的资产配置和风险分析
- **深度分析**: 分析行业表现和市场情绪
- **AI洞察**: 获取AI生成的投资建议和风险评估

## 🔒 安全注意事项

1. **API密钥保护**
   - 不要在公开代码中暴露API密钥
   - 使用环境变量存储敏感信息

2. **数据安全**
   - 所有数据传输使用HTTPS
   - 敏感数据加密存储

## 🛠️ 开发指南

### 添加新的数据源
1. 在 `server/app.js` 中添加新的API调用方法
2. 更新前端界面显示新数据

### 自定义AI提示
修改 `server/app.js` 中的 `generateInvestmentAdvice` 方法，调整AI分析提示词。

### 扩展投资组合模型
在配置中添加新的投资组合配置和风险等级。

## 🐛 故障排除

### 常见问题

1. **API调用失败**
   - 检查网络连接
   - 验证API密钥是否正确
   - 查看API使用限制

2. **数据不更新**
   - 检查缓存设置
   - 验证数据源可用性
   - 查看控制台错误信息

3. **图表不显示**
   - 检查Chart.js库加载
   - 验证数据格式
   - 查看浏览器控制台

## 📈 性能优化

1. **数据缓存**: 实现智能缓存策略，减少API调用
2. **懒加载**: 按需加载图表和组件
3. **压缩**: 使用gzip压缩减少传输大小
