# 智能投资分析专家仪表板 - 项目概览

## 🎯 项目简介

这是一个完整的智能投资分析专家仪表板系统，基于实时数据和AI分析，为投资者提供全面的投资决策支持。

## 📁 项目结构

```
investment-dashboard-expert/
├── index.html                 # 🎨 主仪表板界面
├── js/
│   └── app.js                # 🔧 前端应用逻辑
├── server/
│   └── app.js                # 🚀 后端服务
├── templates/                # 📋 原始静态页面模板
│   ├── Investment Suggestion.html
│   ├── Macro Overview.html
│   ├── Portfolio Management.html
│   └── Prediction.html
├── package.json              # 📦 项目依赖配置
├── start.sh                  # 🐧 Linux/Mac启动脚本
├── start.bat                 # 🪟 Windows启动脚本
├── test.js                   # 🧪 系统测试脚本
├── README.md                 # 📖 详细使用说明
└── PROJECT_OVERVIEW.md       # 📋 项目概览
```

## ✨ 核心功能

### 1. 📊 实时市场数据
- **主要指数**: 标普500、纳斯达克、道琼斯
- **商品价格**: 黄金、原油等大宗商品
- **加密货币**: 比特币、以太坊等主流币种
- **经济指标**: 国债收益率、美元指数、VIX恐慌指数

### 2. 💼 投资组合管理
- **智能配置**: 基于风险偏好的资产配置建议
- **风险评估**: 预期回报、最大回撤、夏普比率
- **具体建议**: 详细的ETF和个股推荐

### 3. 🔍 深度市场分析
- **行业分析**: 各行业板块表现对比
- **技术指标**: RSI、MACD、移动平均线
- **市场情绪**: 恐惧贪婪指数、资金流向分析

### 4. 🤖 AI智能洞察
- **智能建议**: 基于Google AI Studio API的个性化投资建议
- **风险评估**: AI驱动的综合风险评估
- **异常检测**: 实时市场异常情况识别

## 🚀 快速启动

### 方法1: 自动启动（推荐）
```bash

./start.sh

```

### 方法2: 手动启动
```bash

```

## 🔧 技术架构

### 前端技术栈
- **HTML5 + CSS3**: 现代Web标准
- **Tailwind CSS**: 快速UI开发
- **Chart.js**: 数据可视化
- **JavaScript ES6+**: 现代JavaScript特性
- **响应式设计**: 移动端适配

### 后端技术栈
- **Node.js**: 服务器运行环境
- **Express.js**: Web框架
- **Axios**: HTTP客户端
- **CORS**: 跨域支持
- **智能缓存**: 内存缓存优化

### AI集成
- **Google AI Studio API**: 核心AI分析引擎
- **Gemini Pro模型**: 先进的语言模型
- **智能提示工程**: 优化的AI提示词设计

### 数据源
- **Yahoo Finance**: 免费股票和指数数据
- **CoinGecko**: 加密货币数据
- **经济指标**: 国债收益率、美元指数、VIX等

## 📊 API端点

| 端点 | 方法 | 描述 |
|------|------|------|
| `/api/market-summary` | GET | 获取市场数据摘要 |
| `/api/sector-performance` | GET | 获取行业表现数据 |
| `/api/generate-advice` | POST | 生成AI投资建议 |
| `/api/portfolio-metrics/:riskLevel` | GET | 获取投资组合指标 |
| `/api/health` | GET | 健康检查 |

## 🎯 使用流程

1. **启动系统**: 运行启动脚本或手动启动
2. **访问界面**: 浏览器打开 `http://localhost:3000`
3. **刷新数据**: 点击"刷新实时数据"获取最新市场数据

5. **获取建议**: 点击"生成AI投资建议"获取智能分析
6. **制定策略**: 根据建议调整投资策略
