# Notebooks

This directory contains example Jupyter notebooks that demonstrate how to interact with the financial knowledge graph pipeline.

## Available Notebooks

- `financial_graph_demo.ipynb`: End-to-end walk-through that shows how to load data, train the GNN model, and visualise results.

## Usage

Launch Jupyter Lab or Jupyter Notebook from the project root:

```bash
jupyter lab
```

Open the desired notebook inside this folder to explore the workflow step by step.
